# WordHelper
* 使用 Kotlin 编写的单词小助手
* 使用 MVP + RxJava + Room 完成项目功能
* 使用 JobService 监听粘贴板改变事件
* 使用 Window 作为弹出窗口（类似于 Google 翻译弹出效果）
* 使用 RecyclerView，其中包括 Item 的滑动删除及撤销功能
